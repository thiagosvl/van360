# Documentação de Automação de Cobrança e Repasse PIX

Esta pasta contém a documentação técnica detalhada para a implementação da funcionalidade de automação de cobrança dos pais e o repasse instantâneo via PIX para os motoristas na plataforma Van360.

## Conteúdo:

*   **`arquitetura_repasse_pix.md`**: Detalha a visão geral do fluxo, a estrutura de banco de dados necessária e a lógica de backend para a automação de cobrança e repasse PIX.
*   **`refatoracao_repasse_pix.md`**: Descreve as refatorações e novas implementações necessárias no backend (Node.js/TypeScript) e no frontend (React/Next.js) para suportar a funcionalidade de repasse PIX.
*   **`sql_repasse_pix.sql`**: Contém os comandos SQL para criar as novas tabelas e adicionar as colunas necessárias no Supabase para o gerenciamento das cobranças e transações de repasse.
